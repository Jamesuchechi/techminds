CSS Transitions, Animations & Keyframes

Learn how to bring web pages to life using smooth effects, movement, and custom animation steps.

1. CSS Transition

A transition makes a CSS property change smoothly from one value to another. Hover over the box below.

Hover Me 
transition: background 0.5s ease;

2. CSS Animation

Animations run automatically without requiring hover.

Spin 
animation: spin 2s linear infinite;

3. CSS Keyframes

Keyframes define the stages or steps of an animation. 

@keyframes spin{
   from{
      transform: rotate(0deg);
   }

   to{
      transform: rotate(360deg);
   }
}
