HTML Attributes

HTML attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like: name="value".

Common HTML Attributes

  • id: Specifies a unique id for an element.
  • class: Specifies one or more class names for an element (used for CSS and JavaScript).
  • style: Specifies an inline CSS style for an element.
  • title: Provides additional information about an element (displayed as a tooltip).
  • href: Specifies the URL of a link (used in <a> elements).
  • src: Specifies the URL of an image (used in <img> elements).
  • alt: Provides alternative text for an image (used in <img> elements).
  • value: Specifies the value of an input element (used in <input> elements).
  • name: Specifies the name of an input element (used in <input> elements).

Global Attributes

Global attributes can be used on any HTML element. Some common global attributes include:

  • accesskey: Specifies a shortcut key to activate/focus an element.
  • contenteditable: Specifies whether the content of an element is editable.
  • data-*: Used to store custom data private to the page or application.
  • dir: Specifies the text direction for the content in an element (ltr for left-to-right, rtl for right-to-left).
  • draggable: Specifies whether an element is draggable.
  • hidden: Specifies that an element is not yet, or is no longer, relevant.
  • lang: Specifies the language of the element's content.
  • spellcheck: Specifies whether the element is to have its spelling and grammar checked.
  • tabindex: Specifies the tab order of an element.
  • title: Provides additional information about an element (displayed as a tooltip).

Event Attributes

Event attributes are used to specify JavaScript code to be executed when an event occurs. Some common event attributes include:

  • onclick: Executes JavaScript code when an element is clicked.
  • onmouseover: Executes JavaScript code when the mouse pointer moves over an element.
  • onmouseout: Executes JavaScript code when the mouse pointer moves out of an element.
  • onchange: Executes JavaScript code when the value of an element changes.
  • onkeydown: Executes JavaScript code when a key is pressed down.
  • onkeyup: Executes JavaScript code when a key is released.
  • onsubmit: Executes JavaScript code when a form is submitted.
  • onload: Executes JavaScript code when a page or an image has finished loading.
  • onunload: Executes JavaScript code when a page is unloaded.
  • onresize: Executes JavaScript code when the browser window is resized.

HTML Form Attributes

  • action: Specifies where to send the form data when a form is submitted.
  • method: Specifies the HTTP method to use when sending form data (GET or POST).
  • enctype: Specifies how the form data should be encoded when submitting it to the server (used with POST method).
  • autocomplete: Specifies whether the browser should autocomplete the form fields.
  • novalidate: Specifies that the form should not be validated when submitted.
  • target: Specifies where to display the response after submitting the form (e.g., _blank, _self, _parent, _top).
