Semantics is the study of meaning in language, programming, and user interfaces. In HTML, semantic elements communicate structure and purpose, improving accessibility, SEO, and maintainability.

About Semantics

Semantic HTML uses meaningful tags to describe the role of each element, making content easier to read for people and machines.

Semantic elements such as <header>, <nav>, and <article> improve readability and accessibility. They also help search engines and assistive technologies understand your page structure.

When code communicates intent, users enjoy a faster, more intuitive experience and developers benefit from cleaner, easier-to-maintain code.

Semantic HTML Examples

  • <header> — Introductory content or navigation.
  • <nav> — Main navigation links.
  • <article> — Standalone content item.
  • <section> — Thematically grouped content.
  • <footer> — Page or section closing content.
