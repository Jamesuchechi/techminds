HTML Attributes

Attributes provide additional information about HTML elements. They are always written within the opening tag, usually in a `name="value`" format

Go to Header

Example of an Attribute

In the following example, the `href` attribute specifies the URL of the page the link goes to: 

        Visit Example.com

In this example, `href` is the attribute name and `https://www.example.com` is the attribute value. When you click on the link, it will take you to the specified URL.

Common HTML Attributes

Global attributes can be used on any HTML element.

  • id: Specifies a unique id for an element.
  • class: Specifies one or more class names for an element (used for CSS and JavaScript).
  • style: Specifies inline CSS styles for an element.
  • title: Provides additional information about an element (displayed as a tooltip).
  • data-*: Used to store custom data private to the page or application.

There are also specific attributes for certain elements, such as `src` for images, `alt` for alternative text, and `type` for input fields.
